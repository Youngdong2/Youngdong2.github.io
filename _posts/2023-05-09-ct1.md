## 내 풀이
* 리스트 하나를 만들어 계속 비교해가면 된다.
* 이 문제의 핵심은 return하는 [(i%n)+1, (i//n)+1] 이부분인 것 같다.

```python
def solution(n, words):
    stack = [words[0]]
    for i in range(1, len(words)):

        if (stack[-1][-1] == words[i][0]) and (words[i] not in stack):
            stack.append(words[i])
        else:
            return [(i%n)+1, (i//n)+1]
            break


    if len(stack)==len(words):
        return [0, 0]
```
