## 내 풀이
* 해당시점에서 그 시점의 값이 이전에 저장했던 값보다 크다면 업데이트 하는 방식으로 풀었다.

```python
def solution(number, k):
    result = []
    for i in range(len(number)):
        while len(result) != 0 and result[-1] < number[i] and k > 0:
            result.pop()
            k -= 1
        result.append(number[i])

    if k > 0:
        result = result[:-k]
    
    answer = ''.join(result)
    return answer
```
