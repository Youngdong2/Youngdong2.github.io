## 내 풀이
* 이 문제는 스택 자료구조를 사용하면 수월하게 풀 수 있다.

```python
def solution(s):
    result = []
    for i in s:
        if len(result) == 0:
            result.append(i)
        elif result[-1] == i: 
            result.pop()
        else: 
            result.append(i)
    if len(result) == 0: 
        return 1
    else: 
        return 0 
```
