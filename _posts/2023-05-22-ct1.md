## 내 풀이
* 이 문제는 큐를 사용하여 풀었다.
* goal에 있는 단어들이 큐의 맨 앞 인덱스에 있다면 제거하는 형식으로 풀면 된다.


```python
def solution(cards1, cards2, goal):
    from collections import deque

    cards1 = deque(cards1)
    cards2 = deque(cards2)

    for g in goal:
        if cards1 and g == cards1[0]:
            cards1.popleft()
        elif cards2 and g == cards2[0]:
            cards2.popleft()

        else:
            return 'No'
    
    return 'Yes'
```
