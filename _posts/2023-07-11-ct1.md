## 내 풀이
* 필요한 시간의 최소값을 구하려면 걸리는 시간이 적은 시간부터 누적합을 하면 된다.

```python
n = int(input())
li = list(map(int, input().split()))
li.sort()

count = 0
for i in range(1, n+1):
    count += sum(li[:i])
    
print(count)
```
