## 내 풀이
* 이 문제는 먼저 이름과 점수를 매칭하는 딕셔너리를 만들면 편하다.
* 딕셔너리를 이용해 photo에 있는 이름을 검색해 점수를 계산한다.


```python
def solution(name, yearning, photo):
    name_dict = {}
    for i in range(len(name)):
        name_dict[name[i]] = yearning[i]
        
    answer = []
    for i in range(len(photo)):
        score = 0
        for n in photo[i]:
            if n in name_dict.keys():
                score += name_dict[n]
        answer.append(score)
                
    return answer
```
