## 내 풀이
* 참고 길이만큼 elements를 이어붙혀 합을 구하면 편하다.
* 주의할 점은 리스트로 append하면 런타임에러가 나기 때문에 set에 저장을 해야한다.

```python
def solution(elements):
    answer = set()
    for i in range(len(elements)):
        a = elements + elements[:i]
        for j in range(len(elements)):
            value = a[j:j+i+1]
            answer.add(sum(value))
    
    return len(answer)
```
