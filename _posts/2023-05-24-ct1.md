## 내 풀이

* 조금 어려운 문제였다.
* section을 순차적으로 돌며 le를 업데이트해준다.
* le의 업데이트는 m만큼의 크기만큼 커져간다.

```python
def solution(n, m, section):
    answer = 0
    le = 0

    for i in range(len(section)):
        if section[i] <= le:
            le = le

        else:
            answer += 1
            le = section[i]+m-1
            
    return answer
```
