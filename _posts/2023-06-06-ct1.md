## 내 풀이

* 문제에 따라 구현을 하면 되는문제다.
* 목표 문자열을 만들 수 없을 때의 처리가 조금 까다로웠는데 bol 변수를 사용해서 해결했다.

```python
def solution(keymap, targets):
    result = []
    for target in targets:
        count = 0
        for word in target:
            li = []
            bol = False
            for key in keymap:
                if word in key:
                    for i in range(len(key)):
                        if key[i]==word:
                            li.append(i+1)
                            bol = True
                            break
            if bol==False:
                count = -1
                break

            else:
                count += min(li)

        result.append(count)
        
    return result
```
