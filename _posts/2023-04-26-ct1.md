## 내 풀이
* 이 문제는 제한사항에 숫자범위가 작기때문에 3중 for문으로 쉽게 풀 수 있다.

```python
def solution(number):
    count = 0
    for i in range(len(number)):
        x1 = number[i]
        for j in range(1, len(number)-i):
            x2 = number[i+j]
            for k in range(1, len(number)-i-j):
                x3 = number[i+j+k]

                if x1+x2+x3==0:
                    count+=1
                    
    return count
```
