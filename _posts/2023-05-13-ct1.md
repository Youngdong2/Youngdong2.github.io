## 내 풀이
* 이 문제는 규칙을 계속 찾아야 했다. 사실 우연하게 규칙을 찾아는데 규칙을 못찾으면 힘든 문제일 것 같다.
* 검은색칠이 점프, 빨간색칠이 순간이동이라 하면 다음과 같이 표현할 수 있다. 표현하다보면 건전지 사용량이 N을 이진법으로 바꿨을 떄 1의 개수와 같은 것을 알 수 있다.

![fig2]({{site.url}}/images/2023-05-13-ct1/문제풀이.png)

```python
def solution(n):
    ans = bin(n)[2:].count('1')

    return ans
```
