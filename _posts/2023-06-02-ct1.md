## 내 풀이
* 이 문제는 문자열을 문제에  나온 순서대로 진행하면 쉽게 풀 수 있다.


```python
def solution(new_id):
    idd = new_id
    for s in new_id:
        if s.isalpha() or s.isdigit() or s=='-' or s=='_' or s=='.':
            pass
        else:
            idd = idd.replace(s, '')
            
    idd = idd.lower()
    while idd.count('..')!=0:
        idd = idd.replace('..', '.')
        
    if idd[0]=='.':
        idd = idd[1:]
    
    
    if len(idd)!=0 and idd[-1]=='.':
        idd = idd[:-1]
        
    if len(idd)==0:
        idd = 'a'
    
    if len(idd) >= 16:
        idd = idd[:15]

    if len(idd) <= 2:
        x = idd[-1]
        while len(idd) < 3:
            idd += x
            
    if idd[0]=='.':
        idd = idd[1:]
    
    
    if len(idd)!=0 and idd[-1]=='.':
        idd = idd[:-1]
            
    return idd
```
