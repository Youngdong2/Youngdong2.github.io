## 내 풀이
* 이 문제는 규칙을 찾는게 어려웠다. 아래와 같이 2로 나눴을 때의 몫과 나머지의 합의 규칙을 찾을 수 있었다.
![fig2]({{site.url}}/images/2023-05-10-ct1/풀이.png)

```python
def solution(n,a,b):
    answer = 0
    while True:
        a = (a%2) + (a//2)
        b = (b%2) + (b//2)
        answer += 1
        
        if a == b:
            break

    return answer
```
