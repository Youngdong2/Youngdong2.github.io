## 내 풀이
* 스택 리스트를 만든 후 재료를 append하면서 마지막 4개의 원소가 버거와 같은지 계속 확인한다.
* 만약 버거와 같다면 stack을 네번 pop해준다.

```python
def solution(ingredient):
    berger = [1, 2, 3, 1]
    answer = 0
    stack = []
    for i in ingredient:
        stack.append(i)
        if stack[-4:] == berger:
            answer += 1
            for j in range(4):
                stack.pop()
    return answer
```
