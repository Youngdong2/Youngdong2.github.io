## 내 풀이
* 이 문제는 A, B를 다른 방향으로 정렬한 후 같은 인덱스의 값끼리 곱하면 된다.

```python
def solution(A,B):
    answer = 0
    A = sorted(A)
    B = sorted(B, reverse=True)
    for i in range(len(A)):
        answer += A[i] * B[i]

    return answer
```
