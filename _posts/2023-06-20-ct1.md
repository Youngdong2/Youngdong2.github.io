## 내 풀이
* 스택을 활용하면 어렵지 않게 풀 수 있다.
* 괄호의 종류에 따라 스택의 마지막 원소와 비교하여 괄호가 완성되면 pop을 해주는 방법으로 체크를 한다.

```python
def check(s):
    stack = []
    for i in s:
        if len(stack)==0:
            stack.append(i)
        else:
            if i==')' and stack[-1]=='(':
                stack.pop()
            elif i==']' and stack[-1]=='[':
                stack.pop()
            elif i=='}' and stack[-1]=='{':
                stack.pop()
            else:
                stack.append(i)
                
    if len(stack)==0:
        return True
    else:
        return False


def solution(s):
    answer = 0
    for i in range(len(s)):
        shift_s = s[i:]+s[:i]
        if check(shift_s):
            answer += 1
            
    return answer
```
