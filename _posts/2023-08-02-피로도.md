## 내 풀이
* 이 문제는 완전탐색문제이다.
* itertools의 permutations를 사용해 던전의 모든 조합을 확인한다.

```python
def solution(k, dungeons):
    from itertools import permutations
    answer = 0
    for p in permutations(dungeons, len(dungeons)):
        hp = k
        count = 0

        for pm in p:
            if hp >= pm[0]:
                hp -= pm[1]
                count += 1

            if count > answer:
                answer = count
                
    return answer
```
