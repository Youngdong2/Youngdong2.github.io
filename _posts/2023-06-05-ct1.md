## 내 풀이
* 이런 문제는 문자열을 아키코드로 바꾼 후 연산을 해주면 편한 것 같다.

```python
def solution(s, skip, index):
    answer = ''
    for a in s:
        count = 0
        word = ord(a)
        while count!=index:
            word += 1
            if word > ord('z'):
                word = word - ord('z') + ord('a') - 1

            if word in [ord(w) for w in skip]:
                continue
            count += 1

        answer += chr(word)
        
    return answer
```
