## 내 풀이

* 가로, 세로를 나눠 #의 인덱스를 저장한 후 최솟값과 최댓값을 리턴하면 되는 문제이다.

```python
def solution(wallpaper):
    raw = []
    col = []
    for i in range(len(wallpaper)):
        for j in range(len(wallpaper[i])):
            if wallpaper[i][j] == '#':
                raw.append(i)
                col.append(j)
                
    return [min(raw), min(col), max(raw)+1, max(col)+1]
```
