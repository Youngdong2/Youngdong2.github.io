## 내 풀이
* 이 문제는 재귀함수로 풀면 런타임 에러가 뜬다.
* 따라서 몫까지 미리 계산해서 리스트에 넣은 후 마지막 값을 리턴해야한다.

```python
def solution(n):
    li = []
    for i in range(n+1):
        if i==0 or i==1:
            li.append(i)
        else:
            li.append((li[i-1]+li[i-2])%1234567)
    answer = li[-1]
    return answer
```
