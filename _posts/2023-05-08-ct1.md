## 내 풀이
* 문제를 따라 코드를 짜면 그리 어렵지 않은 문제이다.

```python
def solution(a, b, n):
    answer = 0
    while n >= a:
        x = n % a
        n = (n//a) * b
        answer += n
        n += x
    return answer
```
