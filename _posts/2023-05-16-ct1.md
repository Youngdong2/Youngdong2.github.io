## 내 풀이
* 이 문제는 월별 날짜를 정의하면 쉽게 풀 수 있다.


```python
def solution(a, b):
    week_day = ['THU','FRI','SAT','SUN','MON','TUE','WED','THU']
    month_day = [31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]
    
    day = sum(month_day[:a-1])+b
    
    return week_day[day%7]
```
