## 내 풀이
* 이 문제는 경우의 수를 찾아가며 규칙을 찾으면 피보나치 수열인 것을 알 수 있다.
* 하지만 피보나치 수열을 재귀함수로 구현하면 런타임 에러가 나기 때문에 다음과 같이 구현했다.


```python
def solution(n):
    if n == 1:
        return 1 % 1234567
    elif n == 2:
        return 2 % 1234567
    else:
        a,b = 1,2
        for i in range(n-2):
            a, b  = b, a+b
        return b % 1234567
```
