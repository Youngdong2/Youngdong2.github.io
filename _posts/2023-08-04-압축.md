## 내 풀이
* 먼저 대문자 알파벳의 사전을 만든다.
* 이 문제의 핵심은 index를 업데이트 해주는 것 같다.
* while문을 돌며 사전에 해당 알파벳이 있다면 index를 업데이트 해주고, 없다면 check에 그 다음 문자열을 넣어주는 식으로 해결했다.

```python
def solution(msg):
    dic = {}
    for i in range(26):
        dic[chr(65+i)] = i+1

    answer = []
    index = 0
    check = ''
    num = 27

    while index < len(msg):
        check += msg[index]

        if check in dic:
            index += 1

        else:
            dic[check] = num
            num += 1
            answer.append(dic[check[:-1]])
            check = ''

    answer.append(dic[check])
    return answer
```
