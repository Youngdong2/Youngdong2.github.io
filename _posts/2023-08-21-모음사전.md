## 내 풀이

* DFS로 풀수있는 완전탐색 문제이다.
* 먼저 check에 현재 문자를 추가한다.
* 그리고 alpha 리스트를 반복하며 문자열 s의 길이가 5보다 작다면, 문자열에 alpha[i]를 추가해 다시 DFS를 돈다.

```python
def dfs(s, check, alpha):
    check.append(s)
    for i in alpha:
        if len(s) != len(alpha):
            dfs(s+i, check, alpha)
        else:
            s[:-1]

def solution(word):
    alpha = ['A', 'E', 'I', 'O', 'U']
    check = []
    for i in range(len(alpha)):
        dfs(alpha[i], check, alpha)

    return check.index(word)+1
```
