## 내 풀이
* 3을 빼가며 5로 나누어지는지 확인하고 5의 배수가 되면 나누는 식으로 해결할 수 있다.

```python
n = int(input())
def sugar(n):
    count = 0
    while n>=0:
        if n%5==0:
            count += n//5
            break
        n -= 3
        count += 1
    else:
        count = -1
    return count

print(sugar(n))
```
