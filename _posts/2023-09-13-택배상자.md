## 내 풀이

* 보조벨트에 메인벨트의 물품을 올리고, order와 맞으면 꺼내고 아니면 계속 쌓는 식으로 풀면 해결할 수 있다.

```python
def solution(order):
    temp = []
    i = 1
    count = 0

    while i!=len(order)+1:
        temp.append(i)
        while temp and temp[-1]==order[count]:
            count+=1
            temp.pop()

        i+=1
        
    return count
```
