## 풀이
* 다리라는 스택을 만든 뒤 들어와야 할 트럭이 무게를 초과하면 현재 올라와있는 트럭을 보낸다.
* 반복문을 돌때마다 1초씩 더해준다.


```python
def solution(bridge_length, weight, truck_weights):
    answer = 0
    bridge = [0] * bridge_length

    while len(bridge):
        answer += 1
        bridge.pop(0)
        if truck_weights:
            if sum(bridge) + truck_weights[0] < weights:
                bridge.append(truck_weights.pop(0))
            else:
                bridge.append(0)
                
    return answer
```
